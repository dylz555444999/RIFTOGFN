# ⌨️ Console

Open the console using:

* `~` (Tilde)
* `F8`

---

# 🎆 Events

Start the current event:

```console
cheatscript StartEvent
```

---

# 🚗 Vehicles

Spawn a vehicle:

```console
cheatscript SpawnVehicle <VehicleID>
```

| Vehicle       | Command                 |
| ------------- | ----------------------- |
| Mech          | TestMechVehicle         |
| ATK           | GolfCartVehicleSK       |
| Driftboard    | JackalVehicle_Athena    |
| Baller        | OctopusVehicle          |
| Quad Crasher  | AntelopeVehicle         |
| Plane         | FerretVehicle           |
| Cannon        | PushCannon              |
| Shopping Cart | ShoppingCartVehicleSK   |
| Surfboard     | SurfboardVehicle_Athena |

Example:

```console
cheatscript SpawnVehicle GolfCartVehicleSK
```

# 🎒 Items

Spawn a pickup:

```console
cheatscript SpawnItem <ItemID>
```

Give yourself an item:

```console
cheatscript GiveItem <ItemID>
```
